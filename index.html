Почему серии не работают

1. Информационная бомбардировка. 3-5 сообщений подряд без паузы — клиент не успевает вовлечься.

2. Нет точки ответа. Когда всё уже сказано — клиент не знает, на что реагировать.

3. Ощущение монолога. Вместо диалога клиент получает лекцию → закрывает, «потом».

Как надо

1. Одно сообщение = одна мысль. Вопрос → пауза → ответ → следующий вопрос.

2. Вопрос в конце. «Отправила презентацию! Какая площадка ближе?»

3. Документы отдельно. Файл → пауза → короткий вопрос.

13. Рекомендации

Для тёплых лидов

1Ответ за 5 минут. Тёплый лид = деньги на столе. Клиент W2 ответили через 3 часа — это слишком.
2Follow-up через 48ч если молчит. Клиент W3 потеряна — «планируем поехать на 2 смены» = до 204К₽.
3Не отправлять всё сразу. Один файл → вопрос → пауза. Даже тёплый может «утонуть» в документах.

Для холодных лидов

1Вести диалог, а не монолог. Макс. 150 символов. Одна мысль = одно сообщение. Вопрос в конце каждого.
2Системный follow-up: 48ч / 5дн / 14дн. Сейчас follow-up = только разовая акция перед дедлайном 28.02.
3«Когда планируете решение?» — добавить в скрипт. Без этого вопроса непонятно, когда напоминать.
4Не переводить в ТГ до квалификации. Сначала выяснить потребность, потом — удобный канал.

Определение «тёплый» vs «холодный»

Тёплый = клиент пришёл с решением купить. Первое сообщение содержит: «забронировать», «записать», «планируем поехать», заполнение формы бронирования, конкретный запрос на занятия.
Холодный = решение не принято. «Сколько стоит?», «какие варианты?», «пришлите информацию».
Не лид = спам, исходящая рассылка, запрос на вожатую, системные сообщения.
